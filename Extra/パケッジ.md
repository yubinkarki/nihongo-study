### タバコの箱に書いてある説明

```
たばこの煙は、子供の健康にも悪影響を及ぼします。  
たばこの誤飲を防ぐため、乳幼児の手が届かない所に保管・廃棄を。  
「extra lights」の表現は、健康への悪影響が他製品より小さいことを意味するものではありません。  
```

煙 → けむり  
子供 → こども    
健康 → けんこう    
悪影響 → あくえいきょう → bad influence  
及ぼす → およぼす → to exert・to have an effect on  
誤飲 → ごいん → accidental ingestion  
防ぐ → ふせぐ → to prevent  
乳幼児 → にゅうようじ → infants and toddlers  
届かない → とどかない → cannot reach  
保管・廃棄 → ほかん・はいき → store or dispose  
表現 → ひょうげん → expression  
他製品 → たせいひん → other products  
意味 → いみ → meaning

### 餃子

```
お肉と野菜の旨味を包み込みました。  
```

野菜 → やさい  
旨味 → うまみ → savory flavor  
包み込み → つつみこみ → to wrap・enclose fully  
豚肉 → ぶたにく  
餃子 → ぎょうざ → dumplings  
業務用 → ぎょうむよう → for commercial use  
要冷凍 → ようれいとう → required freezing
