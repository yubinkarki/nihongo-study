## ながら

- Doing verb 1 while doing verb 2
- Use with verb’s stem form

### Examples:
> A: 笑いながら面白い番組を見ました。

> A: アルバイトをやりながら大学で勉強します。

> A: 本を書きながら英語を教えました。

## ています

- Explaining habitual actions

### Examples:

```
A: 毎朝ご飯を食べる前にジョギングをしています。

A: 土日は洗濯しています。

A: 毎晩ビールを飲んでいました。
```

## Verb/Adjective し

- Describing something using more than 1 adjective
- Verb る/ない/た/なかった + し
- い形容詞 + し
- な形容詞だ + し

### Examples:

```
A: 姉のアパートは新しいし、庭もあります。

A: 好きな人はやさしいし、きれいだし、人気があります。

A: 彼女は英語が話せるし、熱心だし、経験もあります。

A: 駅からも近いし、このマンションを買います。
```

## それに

- Besides/Moreover

### Examples:

```
A: どうしてこのマンションを選んだんですか。
B: ジムもあるし、安いし、それに駅からも近いです。　
```

## それで

- Then/So/That’s Why

### Examples:

```
A: 彼女は英語が話せるし熱心だし、経験もあります。それで社長はAさんを選びました。
```

## 助詞のに

- Replacement of the particle へ
- Can be used with these verbs instead of へ
- 行きます、来ます、帰ります、出張します
