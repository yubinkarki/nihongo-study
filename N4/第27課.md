## Potential Verbs 可能形

- Verbs expressing the ability to be able to do something

### Conversion:

For group 1 verbs, change to え sound

- `書きます` ➜ `書けます` ➜ `書ける`
- `買います` ➜ `買えます` ➜ `買える`
- `置きます` ➜ `置けます` ➜ `置ける`
- `読みます` ➜ `読めます` ➜ `読める`

For group 2 verbs, add られ/れ on the stem form

- `食べます` ➜ `食べられます` ➜ `食べられる`
- `起きます` ➜ `起きられます` ➜ `起きられる`
- `借ります` ➜ `借りられます` ➜ `借りられる`
- `足ります` ➜ `足りられます` ➜ `足りられる`

For group 3 verbs, see below

- `来ます` ➜ `来られます` ➜ `来られる`
- `します` ➜ `できます` ➜ `できる`

### Examples:

```
A: 私は英語が話せます。

A: 一人で空港へ行けますか。
```

## 見えます & 聞こえます

- Involuntary potential verbs for *to be seen* & *to be heard*
- Different from their voluntary counterparts 見られます & 聞けます

### Examples:

```		
A: 明日からその映画が見られます。

A: この町から富士山が見えます。

A: オーディオガイドで説明が聞けます。

A: あの部屋からふしぎな音が聞こえます。
```

## できます

- Come into being or be completed or be made

### Examples:

```
A: 学校の近くに新しいモールができました。

A: 晩ご飯ができました。
```

## Particle は

- には、では、 へは
- Used to highlight a noun as a topic

### Examples:

```
部屋にはテレビがあります。

店では酒が買えます。

このビルからは花火がみえます。
```

## Particle も

- Also

### Examples:

```
サラダを食べました。ケーキも食べました。

ナルトさんは英語が話せます。中国語も話せます。

部屋にはテレビがあります。リビングルームにもテレビがあります。
```

## Particle しか

- Only/Just
- Opposite of だけ
- Used with negatives only

### Examples:

```
ボールペンは一個しかありません。

彼女にしか買ってあげない。
```
