## カタオモイ by Aimer

> ⭐️ 例えば君の顔に昔より
>
> ```
> 例 ➜ たと・レイ ➜ example, custom
> 君 ➜ きみ・クン ➜ you, mister, buddy
> 顔 ➜ かお・ガン ➜ face, expression
> 昔 ➜ むかし・セキ・シャク ➜ old times, antiquity
> ```

> ⚡️ シワが増えてもそれでも良いんだ

```
皺 ➜ しわ・しぼ・シュウ・スウ ➜ wrinkle, crease
増 ➜ ます・まし・ふえる・ふやし・ゾウ ➜ increase, add
良 ➜ よい・いい・リョウ ➜ good, pleasing 
```

> 🍐 僕がギターを思うように弾けなくなっても

```
僕 ➜ しもべ・ボク ➜ me, servant
思 ➜ おもう・おもえらく・おぼす・シ ➜ think
弾 ➜ ひく・はずむ・たま・はじく・はじける・ただす・はじきゆみ・タン ➜ bullet, flip, snap
```

> 🥎 心の歌は君で溢れているよ

```
心 ➜ こころ・シン ➜ mind, heart, spirit
歌 ➜ うた・うたう ➜ song, sing
溢 ➜ こぼれる・あふれる・みちる・イツ ➜ overflow, spill
```

> 🧿 高い声も 出せずに 思い通り歌えない

```
高 ➜ たか・たかまる・たかめる・コウ ➜ tall, high, expensive
声 ➜ こえ・こわ・セイ・ショウ ➜ voice
出 ➜ でる・だす・シュツ・セイ ➜ exit, leave, come out, put out
通 ➜ とお・とおり・とおす・とおし・かよう・ツウ・ツ ➜ traffic, pass through, avenue, commute
```
